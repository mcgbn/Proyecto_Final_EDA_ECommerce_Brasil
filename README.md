Proyecto final de Data Analytics donde se realiza un análisis de documentos de un e-commerce.

## Objetivo:
Localizar insights de las distintas operativas del negocio, desde productos a clientes pasando por logística de envío, incluyendo un dashboard interactivo en Power BI

### Primer paso de **exploración inicial** (carpeta notebooks):
- En todos los documentos (carpeta raw)
- Comprobación de nulos y duplicados
- Comprobación de formatos
- Unificación de nomenclatura en la escritura, tanto en filas como en titulos de columna, asegurando que no haya errores
      
### **Limpieza inicial** de datasets (carpeta notebooks):
- Gestión de nulos y duplicados
- Cambio de formatos que no se corresponden a las columnas
- Homogeneización textos
- Exportación de archivos limpios (carpeta limpios)

### **Limpieza avanzada** (carpeta notebooks):
- Unificamos todas las tablas limpias en una.
- Gestión de nulos y duplicados resultantes de la unión
- Cambios finales a partir del dataframe final: creación de categorías y calculos para un análisis más exhaustivo.
- Análisis principal, ayudandonos de visualizacion para guiarnos en el enfoque para la creación el dashboard final.
- Exportamos el documento final unificado con las categorías para su uso para crear el dashboard en Power BI (final unificado)

### Creacion de **dashboard interactivo** en Power BI (carpeta Dashboard Power BI):
- Transformaciones finales, medidas personalizadas.
-  Resumen ejecutivo.
- Analisis por áreas de negocio.
- Presencia de filtros de categoría y temproales para permitir una profundidad de análisis mayor.
